# 我们的恋爱秘密基地 💕

一个专属于你们的浪漫网站，记录美好时光。

## 功能

- **首页**：全屏背景图、渐变遮罩、玻璃拟态卡片，点击任意处进入内页
- **内页**：欢迎区域与卡片展示
- **后台**：更换首页背景图（支持链接或本地上传）

## 使用方式

1. 用浏览器打开 `index.html` 即可访问首页
2. 点击页面任意处进入第二页
3. 访问 `admin.html` 进入管理后台，首次使用需设置密码（至少4位）
4. 在后台可粘贴图片链接或拖拽上传本地图片更换背景

## 部署

可直接将整个文件夹部署到任意静态托管（如 GitHub Pages、Vercel、Netlify 等），或本地用 `npx serve .` 运行。
